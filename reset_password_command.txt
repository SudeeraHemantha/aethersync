# Reset Password Command Reference

If you need to reset a user's password in the future, open your command prompt or terminal in this project folder and run the following command.

## Command

```bash
py -c "import sys; sys.path.append('backend'); import database, sqlite3; h = database.hash_password('NEW_PASSWORD'); conn = sqlite3.connect('aethersync.db'); conn.execute('UPDATE users SET password_hash = ? WHERE username = ?', (h, 'USERNAME')); conn.commit(); conn.close(); print('Password Reset Successful!')"
```

## Example (Resetting 'sharly' to 'sharly123')

```bash
py -c "import sys; sys.path.append('backend'); import database, sqlite3; h = database.hash_password('sharly123'); conn = sqlite3.connect('aethersync.db'); conn.execute('UPDATE users SET password_hash = ? WHERE username = ?', (h, 'sharly')); conn.commit(); conn.close(); print('Password Reset Successful!')"
```

Simply replace `NEW_PASSWORD` and `USERNAME` in the first command template with your choices and press Enter.
